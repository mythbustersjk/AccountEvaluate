## Environmental preparation
Python version：3.10  
Python library： colorama
## Installation
### 1.Clone this repository
```
git clone https://github.com/mythbustersjk/AccountEvaluate.git
```  
### 2.Run main.py 
### If you want to build executable programs, you can use pyinstaller to build.
*install pyinstaller*  
```
pip insrall pyinstaller
```  
*build executable programs*  
```commandline
cd ~\AccountEvaluate
pyinstaller -F main.py
```
*You can find the main.exe file in '~\AccountEvaluate\dist\main', when you first run main.exe, you need set initialization
parameters. After finish setting, all the parameters will save in setting.json.*
